# Audio Asset Notes

The audio files in `CatLogicMansion/GameData/Audio` are original procedural assets generated for Cat Logic Mansion.

## Files

- `move.wav`: soft movement tap.
- `blocked.wav`: short blocked movement cue.
- `undo.wav`: rewind cue.
- `clear.wav`: room clear flourish.
- `mansion_loop.wav`: calm looping background music.

## Source

These files are generated by `Tools/GenerateAudio/generate_audio.py`.

The generator uses only mathematical synthesis:

- sine and triangle oscillators
- ADSR-style envelopes
- deterministic noise
- simple low-pass filtering
- echo-style delay

No third-party samples, loops, recordings, presets, or downloaded assets are used.

## Commercial Use

These generated audio files are project-owned assets for Cat Logic Mansion and are intended for commercial use in this app. Keep this note and the generator script with the project so the asset provenance remains clear.

## Regeneration

Run:

```bash
python3 Tools/GenerateAudio/generate_audio.py
```

This overwrites the WAV files in `CatLogicMansion/GameData/Audio`.
