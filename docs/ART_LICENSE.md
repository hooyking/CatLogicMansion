# Art Asset License

The first-pass game art in `CatLogicMansion/GameData/Art/` is original project artwork generated by this project.

Source script:

```text
Tools/GenerateArt/generate_art.py
```

The generated PNG files are procedural drawings. `art_cat_miso.png` preserves the original GameScene-drawn cat token used before the first PNG art pass, because that shape is the approved in-game character read. The generated PNG files do not use third-party images, stock art, downloaded samples, trademarks, or external copyrighted artwork. They are intended for commercial use in Cat Logic Mansion and can be regenerated or replaced by final commissioned artwork later.
